
Pf7_fws.txt: Fws estimates per sample from Pf7. 

*.clusters.0.25.txt: hmmIBD clonal clusters from population samples. Choose single representative sample from clusters in pf7_filtering. 
pf7_filtering: Choose samples from representative population using Pf7 metadata, Fws information (to filter out complex infections). Use output from hmmIBD in second part of code to filter samples down to cluster representatives.
